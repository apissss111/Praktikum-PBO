# Sistem Jasa Pengelolaan Event Volunteer
Deskripsi Program :
Program ini dibuat menggunakan bahasa Java untuk mengelola data volunteer pada sebuah event. Program menggunakan konsep CRUD (Create, Read, Update, Delete) dengan struktur data ArrayList.

Fitur Program
Program memiliki beberapa fitur utama yaitu:

Create (Tambah Data) Menambahkan data event dan volunteer.

Read (Lihat Data) Menampilkan semua data volunteer yang tersimpan.

Update (Ubah Data) Mengubah data volunteer yang sudah ada.

Delete (Hapus Data) Menghapus data volunteer dari daftar.

Exit Keluar dari program.

Struktur Program
Program terdiri dari dua class:

EventVolunteer.java
Class yang menyimpan atribut data event.

Main.java
Class utama yang menjalankan program dan menu.

Struktur Data
Program menggunakan ArrayList untuk menyimpan data volunteer.

Contoh deklarasi:

ArrayList<EventVolunteer> dataEvent = new ArrayList<>();
