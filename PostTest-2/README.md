## Posttest 2

Pada Posttest 2 program diperbarui dengan menerapkan konsep:

### Encapsulation
Semua atribut pada class EventVolunteer diubah menjadi private agar tidak dapat diakses langsung dari class lain.

### Getter dan Setter
Getter digunakan untuk mengambil nilai atribut.
Setter digunakan untuk mengubah nilai atribut.

### Access Modifier
Program menggunakan dua jenis access modifier:
- private pada atribut
- public pada method
